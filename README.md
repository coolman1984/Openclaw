# OpenClaw Workspace

This repository contains the tools, scripts, and project artifacts built during our work together.

## Included
- `memory-system/` — Operational memory manager v2 and v3 planning
- `memory/` — Human-readable daily memory logs and dashboard
- `dev-assistant.py` — CLI productivity dashboard
- `multi-agent-coding.sh` — Multi-agent coding workflow
- `AGENTS.md` — Agent personalities and usage
- `IDENTITY.md`, `USER.md`, `SOUL.md`, `TOOLS.md` — OpenClaw workspace identity and notes
- `memory-implementation-guide.md`, `memory-system-architecture.md` — Architecture docs

## Excluded from GitHub
- API keys and local auth files
- SQLite databases and runtime caches
- Backup archives and downloaded binaries
- Node modules and temporary install artifacts

## Notes
The repository is curated for portability and safety. Sensitive local state is intentionally ignored by `.gitignore`.
